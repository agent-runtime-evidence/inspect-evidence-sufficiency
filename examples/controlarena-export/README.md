# Synthetic ControlArena export fixture

This directory is a **synthetic** ControlArena trajectory export used only to test the reader. It is hand-authored to model the documented export shape — `trajectory.jsonl`, `tools.json`, `metadata.json` produced by `control-arena trajectory export LOG OUT_DIR` — and is **not** real ControlArena output.

Shape reference: ControlArena (UK AISI / Redwood Research), <https://github.com/UKGovernmentBEIS/control-arena>. Re-check the export schema against the ControlArena version you use before relying on the field mapping.
